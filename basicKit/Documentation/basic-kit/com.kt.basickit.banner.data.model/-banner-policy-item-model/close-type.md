[androidJvm]\

PopupBanner 를 닫을 때의 옵션.

- 
   "close": 닫기. 한번 보여 주고 다시 보여 주지 않음.
- 
   "never": 다시 보지 않기.
- 
   "week": 일주일 동안 보지 않기.
- 
   "today": 오늘은 보지 않기.
